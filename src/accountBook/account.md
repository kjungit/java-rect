# 가계부 만들기 (Collections 활용)

> Java의 인터페이스, 구현 클래스, 그리고 `Map` / `List` 같은 컬렉션을 사용해 간단한 가계부 프로그램을 완성합니다. **아래 Step을 순서대로 따라가면, 마지막 Step에서 프로그램이 완성됩니다.**


---

## 최종 완성 체크리스트

- [x] `Item.java` — 항목(이름, 금액) 클래스 완성
- [x] `AccountBook.java` — 인터페이스에 기능 4개 선언
- [x] `AccountBookImpl.java` — `Map<String, List<Item>>`으로 4개 기능 구현
- [x] `Start.java` — 메뉴 반복 + 입력에 따라 기능 호출 + 종료
- [x] 컬렉션(`Map`, `List`)을 실제로 사용함
- [x] 추가 → 조회 → 내역삭제 → 전체삭제가 순서대로 잘 동작함


- [x] **정렬**: 날짜 목록을 항상 최신순으로 자동 정렬 (`TreeMap` 활용)
- [x] **총 지출 합계**: 모든 날짜의 금액을 합쳐 "이번 달 총 지출" 출력
