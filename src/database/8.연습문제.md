## 1. 조건 조회 (WHERE)

<details>
<summary><b>1)</b> 부서번호가 10번인 부서의 사원번호, 이름, 월급을 출력하시오.</summary>

```sql
SELECT EMPNO, ENAME, SAL
FROM EMP
WHERE DEPTNO = 10;
```

</details>

</details>

<details>
<summary><b>4)</b> 입사일이 1983-01-12인 사원의 이름, 부서번호, 월급을 출력하시오.</summary>

<details>

<details>
<summary><b>9)</b> 이름이 'K'로 시작하는 글자보다 사전순으로 뒤인 이름을 가진 사람의 모든 정보를 출력하시오. (참조: <code>CHAR(ASCII('K') + 1)</code>)</summary>

```sql
FROM EMP
WHERE ENAME > CHAR(ASCII('K') + 1);
```

```sql
SELECT *
FROM EMP
WHERE HIREDATE < '1981-12-09';
```

</details>

<details>
<summary><b>11)</b> 사원번호가 7698보다 작거나 같은 사람들의 사원번호와 이름을 출력하시오.</summary>

<summary><b>21)</b> 이름이 S로 시작하고 마지막 글자가 T인 사람의 모든 정보를 출력하시오. (단, 이름은 전체 5자리)</summary>

## 2. 정렬 (ORDER BY)

<details>
<summary><b>28)</b> 부서번호가 30인 사람을 사원번호 기준 오름차순으로 정렬하시오.</summary>

```sql
SELECT *
FROM EMP
WHERE DEPTNO = 30
ORDER BY EMPNO ASC;
```

</details>

<details>
<summary><b>29)</b> 부서번호가 30인 사람을 사원번호 기준 내림차순으로 정렬하시오.</summary>

```sql
SELECT *
FROM EMP
WHERE DEPTNO = 30
ORDER BY EMPNO DESC;
```

</details>

<details>
<summary><b>30)</b> 부서번호로 오름차순 정렬한 후 급여를 내림차순으로 정렬하시오.</summary>

```sql
SELECT *
FROM EMP
ORDER BY DEPTNO ASC, SAL DESC;
```

</details>

<details>
<summary><b>31)</b> 부서번호 내림차순, 이름 오름차순, 급여 내림차순으로 정렬하시오.</summary>


</details>

## 3. 연산 · 함수

<details>
<summary><b>32)</b> 이름, 급여, 커미션, 급여 총액(급여+커미션)을 출력하시오. (단, 커미션이 NULL인 사람은 제외)</summary>

```sql
SELECT ENAME, SAL, COMM, (SAL + COMM) AS `총액`
FROM EMP
WHERE COMM IS NOT NULL;
```

</details>

<details>
<summary><b>33)</b> 10번 부서의 모든 사람에게 급여의 13%를 보너스로 지불한다. 이름, 급여, 보너스, 부서번호를 출력하시오.</summary>

```sql
SELECT ENAME, SAL, (SAL * 0.13) AS `보너스`, DEPTNO
FROM EMP
WHERE DEPTNO = 10;
```

</details>

<details>
<summary><b>34)</b> 30번 부서의 연봉을 출력하시오. (단, 연말에 급여의 150%를 보너스로 지급)</summary>

```sql
SELECT ENAME, DEPTNO, SAL,
       ((SAL * 12) + (SAL * 1.5)) AS `연봉`
FROM EMP
WHERE DEPTNO = 30;
```

</details>

<details>
<summary><b>35)</b> 부서번호가 20인 부서의 시간당 임금을 출력하시오. (월 근무 12일, 1일 5시간 / 소수점 첫째 자리에서 반올림 / 이름, 급여, 시간당 임금)</summary>

```sql
SELECT ENAME, SAL,
       ROUND((SAL / 12) / 5, 0) AS `시간당 임금`
FROM EMP
WHERE DEPTNO = 20;
```

</details>

<details>
<summary><b>36)</b> 급여가 1500~3000인 사람은 급여의 15%를 회비로 낸다. 이름, 급여, 회비(소수점 둘째 자리에서 반올림)를 출력하시오.</summary>

```sql
SELECT ENAME, SAL,
       ROUND(SAL * 0.15, 1) AS `회비`
FROM EMP
WHERE SAL BETWEEN 1500 AND 3000;
```

</details>

<details>
<summary><b>37)</b> 급여가 2000 이상인 사람은 급여의 15%를 경조비로 낸다. 이름, 급여, 경조비를 출력하시오.</summary>

```sql
SELECT ENAME, SAL,
       (SAL * 0.15) AS `경조비`
FROM EMP
WHERE SAL >= 2000;
```

</details>

<details>
<summary><b>38)</b> 모든 사원의 실수령액(급여에서 10% 세금을 뺀 금액)을 급여가 많은 순으로 이름, 급여, 실수령액을 출력하시오.</summary>

```sql
SELECT ENAME, SAL,
       (SAL - (SAL * 0.1)) AS `실수령액`
FROM EMP
ORDER BY SAL DESC;
```

</details>

<details>
<summary><b>39)</b> 입사일로부터 90일이 지난 날짜를 함께 출력하시오. (이름, 입사일, 90일 후, 급여)</summary>

```sql
SELECT ENAME, HIREDATE,
       DATE_ADD(HIREDATE, INTERVAL 90 DAY) AS `90일 후`,
       SAL
FROM EMP;
```

</details>

<details>
<summary><b>40)</b> 입사일, 입사일로부터 6개월이 지난 날짜, 급여를 출력하시오.</summary>

```sql
SELECT HIREDATE,
       DATE_ADD(HIREDATE, INTERVAL 6 MONTH) AS `6개월 후`,
       SAL
FROM EMP;
```

</details>

## 4. 그룹화 (GROUP BY / HAVING)

<details>
<summary><b>41)</b> 각 부서별 같은 업무를 하는 사람의 인원수를 부서번호, 업무명, 인원수로 출력하시오. (부서번호 오름차순)</summary>

```sql
SELECT DEPTNO, JOB, COUNT(*)
FROM EMP
GROUP BY DEPTNO, JOB
ORDER BY DEPTNO;
```

</details>

<details>
<summary><b>42)</b> 같은 업무를 하는 사람이 4명 이상인 업무와 인원수를 출력하시오.</summary>

```sql
SELECT JOB, COUNT(*)
FROM EMP
GROUP BY JOB
HAVING COUNT(*) >= 4;
```

</details>

<details>
<summary><b>43)</b> 각 부서별 평균 월급, 전체 월급(합계), 최고 월급, 최저 월급을 평균 월급이 많은 순으로 출력하시오.</summary>

```sql
SELECT DEPTNO,
       AVG(SAL), SUM(SAL), MAX(SAL), MIN(SAL)
FROM EMP
GROUP BY DEPTNO
ORDER BY AVG(SAL) DESC;
```

</details>

<details>
<summary><b>44)</b> 급여가 1000 이상인 사원들의 부서별 평균 급여를 출력하시오. (단, 부서별 평균 급여가 2000 이상인 부서만)</summary>

```sql
SELECT DEPTNO, AVG(SAL)
FROM EMP
WHERE SAL >= 1000
GROUP BY DEPTNO
HAVING AVG(SAL) >= 2000
ORDER BY DEPTNO;
```

</details>

<details>
<summary><b>45)</b> 10번 부서에 대해 급여의 평균값, 최대/최소값, 인원수를 출력하시오.</summary>

```sql
SELECT AVG(SAL), MAX(SAL), MIN(SAL), COUNT(*)
FROM EMP
WHERE DEPTNO = 10;
```

</details>

## 5. 조인 (JOIN)

<details>
<summary><b>46)</b> 부서번호, 부서명, 이름, 급여를 출력하시오.</summary>

```sql
SELECT D.DEPTNO, D.DNAME, E.ENAME, E.SAL
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO;
```

</details>

<details>
<summary><b>47)</b> 이름이 'ALLEN'인 사원의 부서명을 출력하시오.</summary>

```sql
SELECT D.DNAME
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO
  AND E.ENAME = 'ALLEN';
```

</details>

<details>
<summary><b>48)</b> 모든 사원의 이름, 부서명, 부서번호, 급여를 출력하시오.</summary>

```sql
SELECT E.ENAME, D.DNAME, D.DEPTNO, E.SAL
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO;
```

</details>

<details>
<summary><b>49)</b> 사원번호, 부서번호, 부서명을 출력하시오. (단, 사원이 근무하지 않는 부서명도 함께 출력)</summary>

```sql
SELECT E.EMPNO, E.ENAME, D.DEPTNO, D.DNAME
FROM EMP E RIGHT OUTER JOIN DEPT D
  ON (E.DEPTNO = D.DEPTNO);
```

> 💡 사원이 없는 부서까지 보여야 하므로, DEPT를 기준으로 하는 `RIGHT OUTER JOIN`을 사용한다.

</details>

## 6. 서브쿼리 (Subquery)

<details>
<summary><b>50)</b> 'ALLEN'의 직무와 같은 사람의 이름, 부서명, 급여, 직무를 출력하시오.</summary>

```sql
SELECT E.ENAME, D.DNAME, E.SAL, E.JOB
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO
  AND E.JOB = (SELECT JOB FROM EMP WHERE ENAME = 'ALLEN');
```

</details>

<details>
<summary><b>51)</b> 'JONES'가 속한 부서의 모든 사람의 사원번호, 이름, 입사일, 급여를 출력하시오.</summary>

```sql
SELECT EMPNO, ENAME, HIREDATE, SAL
FROM EMP
WHERE DEPTNO = (SELECT DEPTNO FROM EMP WHERE ENAME = 'JONES');
```

</details>

<details>
<summary><b>52)</b> 전체 사원의 평균 임금보다 많이 받는 사원의 사원번호, 이름, 부서명, 입사일, 지역, 급여를 출력하시오.</summary>

```sql
SELECT E.EMPNO, E.ENAME, D.DNAME, E.HIREDATE, D.LOC, E.SAL
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO
  AND E.SAL > (SELECT AVG(SAL) FROM EMP);
```

</details>

<details>
<summary><b>53)</b> 10번 부서 사람들 중에서 20번 부서 사원과 같은 업무를 하는 사원의 사원번호, 이름, 부서명, 입사일, 지역을 출력하시오.</summary>

```sql
SELECT E.EMPNO, E.ENAME, D.DNAME, E.HIREDATE, D.LOC
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO
  AND E.DEPTNO = 10
  AND E.JOB IN (SELECT JOB FROM EMP WHERE DEPTNO = 20);
```

</details>

<details>
<summary><b>54)</b> 10번 부서 중에서 30번 부서에는 없는 업무를 하는 사원의 사원번호, 이름, 부서명, 입사일, 지역을 출력하시오.</summary>

```sql
SELECT E.EMPNO, E.ENAME, D.DNAME, E.HIREDATE, D.LOC
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO
  AND E.DEPTNO = 10
  AND E.JOB NOT IN (SELECT JOB FROM EMP WHERE DEPTNO = 30);
```

</details>

<details>
<summary><b>55)</b> 10번 부서와 같은 업무를 하는 사람(10번 부서 제외)의 사원번호, 이름, 부서명, 지역, 급여를 급여가 많은 순으로 출력하시오.</summary>

```sql
SELECT E.EMPNO, E.ENAME, D.DNAME, D.LOC, E.SAL
FROM EMP E, DEPT D
WHERE E.DEPTNO = D.DEPTNO
  AND E.DEPTNO <> 10
  AND E.JOB IN (SELECT JOB FROM EMP WHERE DEPTNO = 10)
ORDER BY E.SAL DESC;
```

</details>

<details>
<summary><b>56)</b> 'MARTIN'이나 'SCOTT'의 급여와 같은 사원의 사원번호, 이름, 급여를 출력하시오.</summary>

```sql
SELECT EMPNO, ENAME, SAL
FROM EMP
WHERE SAL IN (SELECT SAL FROM EMP WHERE ENAME IN ('MARTIN', 'SCOTT'));
```

</details>

<details>
<summary><b>57)</b> 급여가 30번 부서의 최고 급여보다 높은 사원의 사원번호, 이름, 급여를 출력하시오.</summary>

```sql
SELECT EMPNO, ENAME, SAL
FROM EMP
WHERE SAL > (SELECT MAX(SAL) FROM EMP WHERE DEPTNO = 30);
```

</details>

<summary><b>58)</b> 'SMITH'와 같은 부서의 사원명을 출력하시오. (SMITH 본인은 제외)</summary>

```sql
SELECT ENAME
FROM EMP
WHERE DEPTNO = (SELECT DEPTNO FROM EMP WHERE ENAME = 'SMITH')
  AND ENAME <> 'SMITH';
```
